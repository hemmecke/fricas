Citation¶

How to cite FriCAS¶

If you use FriCAS in your research, you should use the following BibTeX entry to refer to FriCAS in any publication.

@Misc{FriCAS,
key = {FriCAS},
author = {{FriCAS team}},
year = {2026},
title = {{FriCAS} d9d27a7638f402e94a7909d98ea79b2131dbf65a—an advanced computer algebra system},
note = {Available at \url{https://fricas.github.io}}
}
