This package implements factorization of polynomials over Z_p using a variant of Kaltofen-Shoup method.

ddfact: (U32Vector, Integer) -> List Record(poly: U32Vector, degree: NonNegativeInteger, separate_factors: () -> List U32Vector)

ddfact(pv, p) computes distint degree factorization of pv. pv is assumed square-free, p is assumed prime.

ini_rdata: (U32Vector, Integer) -> List Record(ind: NonNegativeInteger, poly: U32Vector)

ini_rdata(pv, p) generates data allowing faster computation of remainders from division by pv modulo p. Result is used by other function in this package, in particular red_pol.

mfactor: (U32Vector, Integer) -> List U32Vector

mfactor(pv, p) computes factorization of pv into irreducibles modulo p. pv is assumed square-free, p is assumed prime.

mmul: (U32Matrix, U32Matrix, Integer) -> U32Matrix

mmul(m1, m2, p) multiplies m1 by transpose of m2 modulo p

mod_exp: (U32Vector, Integer, List Record(ind: NonNegativeInteger, poly: U32Vector)) -> U32Vector

mod_exp(pv, d, rd) computes pv^d modulo reduction data in rd.

modular_compose: (U32Vector, U32Matrix, U32Vector, NonNegativeInteger, NonNegativeInteger, List Record(ind: NonNegativeInteger, poly: U32Vector)) -> U32Vector

modular_compose(pol, pm, p1, l1, l2, rd) computes composion of pol with p0 modulo reduction data rd. pm and p1 are as from call to power_matrix(p0, l1, rd). l2 must be such that l1*l2 is bigger or equal to degree from rd.

power_matrix: (U32Vector, NonNegativeInteger, List Record(ind: NonNegativeInteger, poly: U32Vector)) -> Record(matr: U32Matrix, poly: U32Vector)

power_matrix(s, l, rd) computes [m, p] such that p = s^l modulo reduction data rd and column j of m contain s^j modulo reduction data rd. Note that lowest column index is j = 0 and highest index is j=l-1. rd should be obtained by earlier call to ini_rdata.

red_pol: (U32Vector, List Record(ind: NonNegativeInteger, poly: U32Vector)) -> Void

red_pol(pv, rd) computes remainder of pv using reduction data rd produced by earlier call to ini_rdata.
