This package computes charateristi polynomial via interpolation from values at several points. Additionally, it computes determinant of matrix of fractions by clearing common denominators of rows. The determinant routine can be used to compute characteristic polynomial instead of regular determinant.

char_pol_via_den_interpolation: Matrix Fraction R -> SparseUnivariatePolynomial Fraction R if R has GcdDomain

char_pol_via_den_interpolation(m) computes characteristic polynomial of m.

char_pol_via_interpolation: Matrix R -> SparseUnivariatePolynomial R

char_pol_via_interpolation(m) computes characteristic polynomial of m.

det_via_common_denominator: Matrix Fraction R -> Fraction R if R has GcdDomain

det_via_common_denominator(m) computes determinant of m.
